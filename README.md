# JavaScript_Node_Selenium

## Summary
Selenium Page Object Model framework using JavaScript and Mocha.

## Depedencias
### Iniciar Projeto
```
npm init
```

### Install Selenium
```
npm install selenium-webdriver
```

### Install Chromedriver
```
npm install chromedriver
```
```

## Iniciar o teste
```
npm test ./test/homepage.js
```
